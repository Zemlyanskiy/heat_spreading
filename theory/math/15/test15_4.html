Решение сложных систем уравнений.
       Задания для самоконтроля      

     4. На каких положениях основан метод определения устойчивости разностных схем с помощью тестовых задач?

Погрешность решения разностной схемы, аппроксимирующей тестовую задачу, легко определить, поскольку известно истинное решение тестовой задачи
Устойчивость разностных схем, которые могут быть использованы для аппроксимации тестовой задачи, легче проверять с помощью метода гармоник
Разностные схемы, отличающиеся только видом свободных членов (не содержащих искомую функцию), обладают одинаковым типом устойчивости
Разностные схемы, отличающиеся только видом свободных членов (не содержащих искомую функцию), не обладают одинаковым типом устойчивости
